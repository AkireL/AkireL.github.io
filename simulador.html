Simulador de campos vectoriales

Aplicación web para la simulación de escenarios construidos con componentes tridimensionales que estén protegidos por dispositivos creados por el laboratorio de investigación para la protección de regiones de descargas eléctricas atmosféricas y obtener resultados que permitan tomar decisiones.

Roles

Los roles desempeñados: colaborador de análisis y desarrollador. Trabajé colaborativamente con el líder de proyectos.

Contribución

Contribuí en la búsqueda, análisis e implementación de las tecnologías que mejor se adaptaban al desarrollo. El primer paso fue recopilar y analizar los requerimientos recabados, posteriormente buscar las tecnologías para realizar un cuadro comparativo de las ventajas y desventajas de las mismas con el objetivo de presentar las conclusiones y propuestas.
Después de presentar la propuesta, está fue adaptándose acorde a los requerimientos no funcionales. Dentro de la primera etapa se diseño y se comenzo la codificación del sistema web con Python, Django y algunas librerías para el manejo de los datos y generación de reportes; para la persistencia de los datos se usó PostgreSQL. El frontend se compuso de HTML, CSS y jQuery. Para el backend se implementó una API RESTFUL con Django Rest Framework.
Cabe destacar que se requirió modelar los escenarios en Blender con objetos en 3D, específicamente aquellos objetos de protección contra descargas eléctricas, elaborados en el laboratorio de investigación.

Resultados

En la entrega final se cumplió con todos los requerimientos.

Retos

Como primer reto para mí fue aprender un nuevo lenguaje, entender y utilizar un framework asi como sus librerías. No solo fue aprender un nuevo lenguaje o tecnología, sino optimizar el tiempo de respuesta. Dado que para simular las áreas construidas por el usuario y renderizarlas en navegador, este tardaba demasiado tiempo, lo que me obligó a buscar las mejoras del proyecto en cuanto a la optimización, factorización y búsqueda de librerías que me permitieran reducir el tiempo de respuesta.
