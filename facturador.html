Facturador

Proyecto enfocado a la facturación de comprobantes fiscales acorde a la versión 3.3 del Servicio de Administración Tributaria (SAT).

Roles

Los roles desempeñados fueron: investigador, ingeniero de software y desarrollador web.

Contribución

El lenguaje usado fue C# con el framework ASP .Net MVC y para la persistencia de datos SQL Server. Para el frontend se utilizó Boostrap 3 y jQuery.

Resultados

En la entrega final se cumplió con todos los requerimientos.
