Sistema de comisiones

Sistema para la administración de clientes, productos, empleados, cálculo de ventas y pago de comisiones.

Roles

Los roles desempeñados fue analista, diseñador, lider y programador.

Gestión

La gestión se realizó implementado el framework llamado SCRUM. Todos los días realizaba reuniones de 10 a 15 minutos para informarnos de los avances y dudas. Por cada termino de las iteraciones teníamos reuniones remotamente

Conocimientos

La herramienta usada fue Trello para la gestión del proyecto y Bitbucket para alojar el repositorio.

Git para el control de versiones de la aplicación. One Drive para subir todas las evidencias de los defectos reportados por el tester, que estos también eran notificados por Tello.

El lenguaje de programación fue C# con el framework ASP .NET MVC, SQL Server para el manejo de datos, como herramienta de codificación Visual Studio 2019 y para la estilización Boostrap 3.

En este proyecto se realizo manual de usuario y se capacito al usuario final.

Resultados

En la entrega final se cumplió con todos los requerimientos.

Retos

La iniciativa es clave importante para el desarrollo profesional y del proyecto, fue un reto transmitirlo a cada uno de los integrantes.

La comunicación clara y concisa con un lenguaje adecuado es muy importante para que el equipo se sienta motivado y valioso.
