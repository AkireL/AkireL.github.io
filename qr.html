Cuestionarios

El proyecto se compuso de dos aplicaciones: aplicación para la evaluación de cuestionarios de áreas específicas de los hoteles y aplicación para la administración de cuestionario, usuarios, y accesos al aplicable.

Roles

Los roles desempeñados en este proyecto fueron: colaborador de análisis, diseñador de vistas y desarrollador.

Tecnologías

Las tecnologías aplicadas en este proyecto fueron las siguientes: lenguaje de programación PHP y el framework Codeigniter, Boostrap 4 para los estilos de las vistas. Conexión mediante End point para la comunicación de la aplicación de evaluación por cuestionarios.

También programar scrips y tareas cron que se encarguen de enviar reportes por correo electrónico.

Resultados

En la entrega final se cumplió con todos los requerimientos.
