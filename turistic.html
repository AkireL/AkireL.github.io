Turistico

Aplicación web para la simulación de escenarios construidos con componentes tridimensionales que estén protegidos por dispositivos creados por el laboratorio de investigación para la protección de regiones de descargas eléctricas atmosféricas y obtener resultados que permitan tomar decisiones.

Roles

Los roles desempeñados: colaborador de análisis y desarrollador. Trabaje colaborativamente con el líder de proyectos.

Contribución

Contribuí en la búsqueda, análisis e instalación de las tecnologías que mejor se adapten al desarrollo. Apoyé al análisis y codificación de la aplicación. Como primer paso fue recompilar y analizar los requerimientos que se solicitaron, posteriormente buscar las tecnologías y realizar un cuadro comparativo de las ventajas y desventajas de estas. Finalmente presentar conclusiones y propuestas. Después de presentar la propuesta ,fue modificada adaptándose a los requerimientos no funcionales. Dentro de la primera etapa fue diseñar la base de datos y codificación. Para crear escenarios se necesitó modelar en Blender los objectos en 3D, específicamente objectos de protección contra descargas eléctricas que fueron elaborados en el laboratorio de investigación. Durante el desarrollo aprendí y mejoré mis conocimientos en Python, Django y algunas librerías para el manejo de los datos y generación de reportes. Para la persistencia de los datos se usó PostgreSQL. La interacción con el cliente implementamos el framework jquery de javascript. Construir una API rest ful en el backend con Django Rest Framework.

Resultados

En la entrega final se cumplió con todos los requerimientos.
