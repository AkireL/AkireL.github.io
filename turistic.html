Turistico

Este proyecto fue un prototipo para la gestión y venta de paquetes de viajes ofrecidos a público general por medio de colaboradores o vendedores de la empresa.

Roles

Mi rol desempeñado fue desarrollador web.

Contribución

Contribuí en el maquetado y diseño del frontend con HTML, CSS y Bootstrap 3. Colaboré en la codificación de búsquedas dentro de los catálogos usando SQL Server y ASP.NET MVC 5.

Resultados

En la entrega final se cumplió con todos los requerimientos.

Retos y conocimiento adquirido

Durante el desarrollo reforcé un poco mas sobre CSS, especialmente con reglas de media queries y Boostrap 3. Aprendí a usar el framework ASP .NET MVC, LINQ y SQL Server
